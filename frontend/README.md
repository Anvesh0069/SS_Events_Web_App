# SS Events Landing Page

Beautiful, modern landing page for SS Events - Hyderabad's premier event services provider.

## Live Site
Deployed on Vercel

## Features
- Elegant & Vibrant Design
- 3D Animated Gallery
- Mobile Responsive
- WhatsApp Integration
- Professional Contact Section

## Technology Stack
- React 19
- Tailwind CSS
- Lucide React Icons
- React Router

## Local Development
```bash
yarn install
yarn start
```

## Build for Production
```bash
yarn build
```
